C Programs Collection

This repository contains basic C language programs for learning fundamental programming concepts such as loops, characters, and number manipulation.

Programs Included
1. Alphabet Printing Program (Skip 3 Letters)

This program prints alphabets starting from a user-given character and skips the next 3 letters each time using a do-while loop.

How it works:

User enters a starting character.

Program prints the character.

Then it increases the character by 4 positions.

Continues until it reaches 'z'.

Example:
Input: a
Output: a e i m q u y
Code Concept Used:

Character variables

do-while loop

ASCII character increment

2. Count Total Number of Digits

This program counts how many digits are present in a number entered by the user.

How it works:

User enters a number.

Program divides the number by 10 repeatedly.

Each division increases the digit counter.

Loop continues until number becomes 0.

Example:
Input: 12345
Output: digits: 5
Code Concept Used:

while loop

integer division

counter logic

3. Addition of First and Last Digit

This program finds and adds the first and last digit of a number.

How it works:

User enters a number.

Last digit is found using modulo operator %.

First digit is found by dividing number repeatedly by 10.

Both digits are added and displayed.

Example:
Input: 1234
Output: addition of digit: 5

(First digit = 1, Last digit = 4, Sum = 5)

Code Concept Used:

modulo operator %

while loop

number manipulation

Technologies Used

Language: C

Compiler: GCC / Turbo C / CodeBlocks / VS Code

Platform: Windows / Linux / Mac

How to Run the Program

Install any C compiler (GCC recommended)

Save the file with .c extension

Open terminal or command prompt

Compile the program:

gcc filename.c -o output

Run the program:

output

These programs help beginners understand:

Characters and ASCII values

Number manipulation

Basic C programming logic
